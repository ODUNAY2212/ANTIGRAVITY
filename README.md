# ANTIGRAVITY

A static website project built with HTML, CSS, JavaScript, and Tailwind configuration.

## Overview

This project includes a multi-page storefront-style site with pages for home, shop, gallery, checkout, contact, and about.

## Structure

- `index.html` — Main landing page
- `shop.html` — Products and shopping page
- `gallery.html` — Image gallery page
- `checkout.html` — Checkout flow page
- `contact.html` — Contact page
- `about.html` — About page

- `css/style.css` — Core site styles
- `tailwind.config.js` — Tailwind CSS configuration file

- `js/main.js` — Main script logic
- `js/shop.js` — Shop page behavior
- `js/cart.js` — Cart interactions
- `js/checkout.js` — Checkout page behavior
- `js/gallery.js` — Gallery page behavior
- `js/products.js` — Product data and behavior

- `assets/images/` — Image assets used throughout the site

## Usage

Open the HTML files in a browser to preview the site locally. For development with Tailwind, use your preferred build toolchain if one is configured.

## Notes

- This is a static front-end project, so no backend server is required to view the pages.
- If you want to add Tailwind build steps, include the appropriate npm scripts and dependencies.
